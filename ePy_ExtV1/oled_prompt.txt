固定一秒讀取一次 溫濕度，並顯示在 OLED 上 ，溫度顯示在第一行取小數第一位，需要有單位，濕度顯示在第二行，需要有單位%。
當使用者按下 上鍵 的時候，改變顯示的單位 由攝氏 改成 華氏，再按一下 上鍵 改回攝氏

# RL62M02 Mesh device module 
可以使用 class 來建立一個物件，並且使用物件的方法來操作
協助建構一個 mesh device 的物件 
1 .處理 物件的建立 與 UART 的初始化
2. 使用 AT+REBOOT 之後得到的訊息來知道目前device 的狀態 是否已經被綁定了
3. 需要可以 解除綁定 AT+NR 的功能
4. 需要可以 設定自己資料功能 (AT+MDTS) 傳送之前需要確認自身是否處於被綁定的狀態
5. 需要可以接收資料的功能 "MDTSG-MSG" 與 "MDTPG" 兩種
最後建立一個測試class的程式  來測試這個 class 的功能

固定一秒讀取一次 溫濕度，並顯示在 OLED 上 ，
溫度顯示在第一行取小數第一位，需要有單位，濕度顯示在第二行，需要有單位%。
AIN5 讀取 顯示在 OLED 上第三行 不需要單位
當使用者按下 上鍵 的時候，改變顯示的單位 由攝氏 改成 華氏，再按一下 上鍵 改回攝氏
配合mesh device module 的 class 來做一個簡單的 應用
將讀到的 溫度與濕度分兩次 使用 mesh 送出 
 - 溫度 送出格式 "T:xx.x"
 - 濕度 送出格式 "H:xx.x"
 - AIN5 送出格式 "A:xx.x"
